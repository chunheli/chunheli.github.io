免费科学安全上网教程

Posted on 2020-12-23 | Post modified: 2020-12-31 | In tutorial | Views

由于平时有上外网查资料的诉求，但是又不想买VPN，因为既不稳定又比较贵。于是就想着搞个服务器自己搭个梯子来上外网，之前购买过服务器，也要花钱而且网速也很差。后面跟同事了解到谷歌云服务器可以免费试用 3 个月（之前是 1 年…），自己抽空研究了下，发现网速十分给力，看油管 2k 毫无压力，于是就愉快地开启了免费科学安全上网之路。接下来我就介绍下具体该如何操作吧。

前期准备

首先你要有个能够访问外网的环境，注册一个谷歌账户，还要准备一张外币信用卡（比如 visa 信用卡），用来申请谷歌云服务器。为了搭梯子，我直接办了张 visa 信用卡。

试用申请

登陆 https://cloud.google.com/free/ 网站，填写个人信息。国家/地区选择美国，同意服务条款，点继续。

账号类型选择个人，将信用卡信息按要求正确填写，填写后会弹出地址信息。

百度美国地址生成器，随机生成一份地址信息。

将地址信息对应填入，点击开始免费试用。

进入到 console，可以在 Billing 界面看到谷歌赠送的 300$ 和剩余的试用天数。试用申请完毕，接下来我们开始准备创建服务器。

创建防火墙规则

首先我们创建防火墙规则，因为在创建服务器时需要配置防火墙。选择 VPC network 中的 Firewall，点击 CREATE FIREWALL RULE 创建防火墙。

先创建个名字为 in 的防火墙规则（我已经创建过了），设置优先级为 1，将框出来的部分按我的配置配，其余项保持默认，最后点击创建。

再创建个名字为 out 的防火墙规则，注意我文字说明的部分要改一下，然后创建。

创建服务器

点击 Compute Engine 中的 VM instances，点击 CREATE INSTANCE 创建服务器。

为服务器取个名字，然后选择地区，推荐香港，延迟很低，网速杠杠的。机器选择微型，系统选择 CentOS 7，其他保持默认。注意系统不要选错了，否则会导致后续的脚本安装失败。

防火墙全选，并在网络标签处输入刚刚创建的两个防火墙规则的名字，其他保持默认，然后点击创建，一台属于你自己的服务器就这样诞生了。接下来可以在窗口 ping 一下生成的 ip，一般延迟不会超过 30 ms。

接下来要将你的 ip 设置成静态 ip，这个就是梯子服务器的 ip 了。

连接服务器

直接点击这个 ssh 就可以通过浏览器连接到服务器了，不过每次连接都会比较慢，最好可以通过本地终端的 ssh 链接。

需要先网页登录服务器，获取下 root 权限，然后重置下密码，将 /etc/ssh/sshd_config 中 PermitRootLogin no 改为 PermitRootLogin yes，PasswordAuthentication no 改为PasswordAuthentication yes，然后执行 systemctl restart sshd 重启，就可以通过 ssh root@host 的方式用密码登录了。

下载ssr客户端并连接

接下来下载 ssr 客户端，到 https://github.com/shadowsocks 下载对应系统的客户端。下载安装完成后，将 ssr 服务器的地址、端口、密码输入，改个备注，其余保持默认，最好再设置个开机启动。

访问外网测试

打开油管测试下网速吧，2k 视频毫无无压力。
